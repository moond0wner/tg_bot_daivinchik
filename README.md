## О проекте

Telegram-бот, созданный для поиска интересных знакомств. Бот помогает людям находить друг друга, предлагая функционал для создания профилей и поиска совпадений.

## Возможности

*   Создание профиля: Пользователи могут создавать профили с информацией о себе, включая имя, возраст, город, описание и фотографии.
*   Поиск анкет: Возможность поиска анкет по различным критериям (возраст, город и т.д.).
*   Лайки: Пользователи могут выражать симпатию другим профилям.
*   Валидация данных: Валидация введенных пользователем данных (имя, возраст, город и т.д.) для обеспечения корректности и безопасности.
*   Логирование: Логирование всех важных действий бота для упрощения отладки и мониторинга.

## Стек технологий

*   Aiogram: Асинхронный фреймворк для создания Telegram-ботов.
*   Python-dotenv: Для управления переменными окружения.
*   Aiosqlite: Асинхронная библиотека для работы с SQLite.
*   SQLAlchemy: ORM для взаимодействия с базой данных.
*   Pydantic: Для валидации данных и сериализации/десериализации.
*   Pydantic-settings: Для удобного управления настройками проекта.
*   Redis: Используется в качестве хранилища для временных данных.

## Установка

1.  Клонируйте репозиторий:
    ``git clone https://github.com/moond0wner/tg_bot_daivinchik``
    
2.  Установите зависимости:
    ``pip install -r requirements.txt``
    
3.  Создайте файл .env в корне проекта и добавьте необходимые переменные окружения:
   ```
    BOT_TOKEN=ваш_токен_бота
    DATABASE_URL=sqlite+aiosqlite:///db.sqlite3
    REDIS_HOST=localhost
    REDIS_PORT=6379
    DEVELOPER_ID=id_вашего_аккаунта
    
    *   BOT_TOKEN: Токен вашего Telegram-бота.
    *   DATABASE_URL: URL для подключения к базе данных (SQLite, PostgreSQL и т.д.).
    *   REDIS_HOST: Хост Redis-сервера.
    *   REDIS_PORT: Порт Redis-сервера.
    *   DEVELOPER_ID: ID вашего аккаунта.
   ```
## Запуск
```
python main.py
```
